## ğŸ› Troubleshooting

### Menu Doesn't Appear

**Check 1**: Is the script enabled?
- Click Tampermonkey icon
- Find "Tdjs-AutoReg V2"
- Make sure it has a green checkmark

**Check 2**: Are you on the right site?
- Must be `vsphone.com` or `cloud.vsphone.com`
- Won't appear on other sites

**Check 3**: Refresh the page
- Press F5 or Ctrl+R
- Wait 1-2 seconds

**Check 4**: Check console
- Press F12
- Look for "ğŸš€ Tdjs-AutoReg V2 loaded"
- If error, read the message

### Script Won't Install

**Check 1**: Tampermonkey installed?
- Click extensions icon in toolbar
- Look for Tampermonkey
- If not there, install from link above

**Check 2**: Copy all code?
- Make sure you copied the ENTIRE file
- Should start with `// ==UserScript==`
- Should end with `})();`

**Check 3**: Save the script?
- Press Ctrl+S or Cmd+S
- Look for "Script updated" message
- Close and reopen editor

### Referral Link Won't Add

**Check 1**: Correct format?
- Must contain `/invite/`
- Example: `https://www.vsphone.com/invite/abc123`
- No spaces before or after

**Check 2**: Already added?
- Check if it's already in your list
- Script prevents duplicates
